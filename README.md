 Bank Personal Loan Analysis

 📌 Project Overview
This project analyzes customer data from a bank to identify the factors that influence personal loan acceptance. Using SQL for data analysis and Power BI for visualization, the project uncovers patterns in customer demographics, income, education, online banking usage, and credit card ownership to support data-driven decision-making.

🎯 Business Objective
The objective of this project is to help the bank understand which customer characteristics are associated with a higher likelihood of accepting a personal loan. These insights can help improve targeted marketing campaigns and increase loan conversion rates.


 🛠️ Tools Used
- SQL (MySQL)
- Power BI
- Microsoft Excel

 📂 Dataset
The dataset contains customer information, including:
- Customer ID
- Age
- Income
- Family Size
- Education
- Credit Card Ownership
- Online Banking Usage
- Securities Account
- Personal Loan Acceptance

 ❓ Business Questions Answered
1. How many customers are in the dataset?
2. What percentage of customers accepted the personal loan?
3. What is the average income of customers?
4. Which education level has the highest loan acceptance?
5. Does income affect loan acceptance?
6. Which age group accepts loans the most?
7. Does having a credit card influence loan acceptance?
8. Does online banking usage affect loan acceptance?
9. Does family size influence loan acceptance?

 📊 Files Included
- **Bank_Personal_Loan.csv** – Dataset
- **SQL_Queries.sql** – SQL queries used for the analysis
- **Bank Loan Analysis.pbix** – Power BI dashboard
- **screenshots/** – Dashboard screenshots


 💡 Key Insights
- The percentage of customers who accepted the personal loan was determined using SQL.
- Customers with higher average incomes were more likely to accept personal loans.
- Loan acceptance varied across education levels.
- Credit card ownership showed differences in loan acceptance rates.
- Online banking usage was analyzed to determine its relationship with loan acceptance.
- Family size and age groups were also examined to identify customer segments with higher loan acceptance.
